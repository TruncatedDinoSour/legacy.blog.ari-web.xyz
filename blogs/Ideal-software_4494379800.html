Ideal software

2021-12-06 20:42:58 EET | back | home | git

The ideal software to me is minimal , it is not bloated, full of crammed "features" I will not use, even if it lacks features I want to have the ability to build my own features so if it lacks something or has any useless feature I could remove the feature . The software should be small , so I could easily edit it and so -- should have no creepiness and should be FOSS It should not be owned by any company except the community and give the user full control .
